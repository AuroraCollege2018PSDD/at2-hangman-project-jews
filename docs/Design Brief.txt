What The Client Wants:

The client wants a simple game the involves death. This is to relax her child
when he is bored. The client would like the game in 6-7 weeks and also wants 
it to be simple so that it can run on a laptop. No offensive words can be 
used and it has to be quick or interesting.

Proposals:

The game will start off simple and in text form. As we are making the game 
we will develop it become more graphic. We will need the game to be easy to 
use as it is a child that will be using it. It will contain no offensive words. 
Simple but also quick and interesting.
